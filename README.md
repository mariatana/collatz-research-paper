Collatz Conjecture Exploration
This project is a small dive into one of the most famous unsolved problems in mathematics: the 3n + 1 conjecture. I built this to visualize how numbers behave under these rules and to practice my Python skills.

Why this project?
As a Computer Science student with a solid foundation in Linear Algebra and Analysis, I’ve always been fascinated by how simple recursive rules can create such complex patterns. This was a great way to bridge my math background with actual code.

What it does
The Logic: Implements the basic Collatz sequence: if even, divide by 2; if odd, multiply by 3 and add 1.

Visualization: Generates plots to show the path every number takes before eventually hitting the 4-2-1 loop.

Teamwork: Developed this as a collaborative effort, using Git to manage our progress and keep the code organized.

Tech Stack
Language: Python

Libraries: Matplotlib (for the graphs)

Version Control: Git

How to run it
Just make sure you have Python and Matplotlib installed, then run:

Bash
python collatz.py
